## 🚨 **IF SOMETHING GOES WRONG:**

### Build Failed?
```
Solution: Check Netlify build logs
Deploys → Failed build → View log
Common issues: Missing base directory, wrong build command
```

### Site shows 404?
```
Solution: Check publish directory = "dist"
Most common mistake: Wrong publish folder
```

### API calls failing?
```
Solution: Set VITE_API_BASE_URL environment variable
Redeploy after adding
```

### Routes not working?
```
Solution: Already configured in netlify.toml
If still broken: Check Netlify redirects rules
```
